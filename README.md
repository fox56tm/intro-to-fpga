# Intro to FPGA with Clash

A repository for completing an FPGA programming course. The curriculum covers the progression from basic bitwise operations in [SystemVerilog](https://chipverify.com/tutorials/systemverilog) to describing relatively non-trivial logic using the [Clash language](https://clash-lang.org/).

Course materials: https://github.com/Lamagraph/intro-to-fpga-with-clash

Part of the 2026 Summer Programming School of the Faculty of Mathematics and Mechanics at Saint Petersburg State University.
